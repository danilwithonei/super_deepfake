### install
```bash
sudo apt-get install -y libxcb-cursor-dev
pip install -r requirements.txt
```
### before run

if you have Linux then:
1. `sudo modprobe v4l2loopback`

if you have Windows then:
1. my condolences.
2. install `OBS`.

### Run
```bash
python main.py
```
